# just-ok-claude-stuff

All the files in .claude/agents, .claude/commands, and .claude/skills contain generic development workflow instructions that are:

Agents (5 files)

- Generic descriptions of security auditing, test writing, documentation, Vite optimization, and meta-generation
- Standard software development methodologies applicable to any Vue/TypeScript project
- No business logic, API secrets, credentials, or proprietary algorithms

Skills (5 files)

- Coding conventions: Vue patterns, TypeScript style, CSS approach, Nx workspace structure
- Project philosophy: basically "keep it simple" principles
- These are team preferences, not trade secrets

Commands (13 files)

- Workflow shortcuts for: linting, PR reviews, refactoring, E2E testing, security checks
- Generic prompts telling Claude how to analyze code
- No proprietary business rules or sensitive data
